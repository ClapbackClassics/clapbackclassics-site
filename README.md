# clapbackclassics-site
TikTok API verification + App Info
This page provides information required for TikTok’s Content Posting API.
  The Clapback Classics automation app posts content to TikTok on behalf of the authorized account.
  No user data is collected or stored.
Terms of Service
  This app posts content to TikTok using the official TikTok Content Posting API.
  All actions require explicit authorization from the TikTok account owner.
  No data is shared with third parties.
Privacy Policy
  This app does not collect personal information.
  No data is stored on external servers.
  Permissions are used solely to publish content to TikTok.
